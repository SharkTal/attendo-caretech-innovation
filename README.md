## 🇫🇮 Suomeksi: Innovaatioehdotus hoivatyön kirjaamiseen

Tämä projekti on käyttäjälähtöinen kehitysehdotus **Efima Puhekirjaus** -järjestelmän optimoimiseksi Attendon hoivaympäristössä.

### Tausta (The Discovery)
Nykyinen järjestelmä on mullistanut kirjaamisen, mutta se toimii edelleen yksilökeskeisellä logiikalla. Ryhmätilanteissa, kuten **jumpassa** tai **ulkoilussa**, hoitaja joutuu toistamaan saman kuvauksen jokaiselle asukkaalle erikseen.

### Ratkaisu (The Solution)
Ehdotan **​"Yksi kirjaus, useampi osallistuja"​** -mallia:
1. **Master Record:​** Hoitaja sanelee aktiviteetin sisällön kerran.
2. **Multi-Selection:​** Käyttäjä valitsee kaikki osallistuneet asukkaat.
3. **Automated Distribution:​** Järjestelmä hajauttaa merkinnän itsenäisiksi kirjauksiksi jokaiselle asukkaalle taustalla.

## 🇺🇸 English: Edge Case Discovery in Care Documentation

As a healthcare professional with an **IT background**, I identified a significant "Edge Case" in the current AI-assisted documentation workflow at Attendo.

### The Problem
Redundant manual repetition during group activities. If 10 residents participate in a session, the caregiver repeats the same voice command 10 times.

### The Impact
By shifting from a **Person-Centric** to an **Activity-Centric** documentation model for specific scenarios, we can:
- Reduce repetitive tasks by **90%​**.
- Maintain **100% Audit Compliance**.
- Give more time back to **genuine care and encounters**.
